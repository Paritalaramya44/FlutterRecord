Week 3 — Android Studio Setup & Flutter Widgets

Date: 07-08-2025

Learning Objectives

  1. Install Android Studio in Ubuntu using different methods
  2. Create and run Dart programs in Android Studio
  3. Understand and implement Flutter widgets: Text, Image, and Container
  4. Display text and images using Container widget

1. Installing Android Studio in Ubuntu

Method 1: Install via Snap (Recommended for Beginners)

This is the simplest method for installing Android Studio.

Step 1: Open Terminal (Ctrl + Alt + T)

Step 2: Run this command:

sudo snap install android-studio --classic

This will automatically download and install the latest Android Studio.

Method 2: Manual Installation from Official Site

Step 1: Install Java (Required)

sudo apt update
sudo apt install openjdk-17-jdk

Step 2: Download Android Studio

Go to: https://developer.android.com/studio

Download the Linux .tar.gz file, or use terminal:

wget https://redirector.gvt1.com/edgedl/android/studio/ide-zips/2023.2.1.21/android-studio-2023.2.1.21-linux.tar.gz

Step 3: Extract the file

tar -xvzf android-studio-*.tar.gz

Step 4: Move to /opt

sudo mv android-studio /opt/

Step 5: Launch Android Studio

/opt/android-studio/bin/studio.sh

Follow the setup wizard to install SDK, configure emulator, etc.

Optional: Create Desktop Shortcut

To launch it from the app menu:

sudo nano /usr/share/applications/android-studio.desktop

Paste the following content:

[Desktop Entry]
Version=1.0
Type=Application
Name=Android Studio
Exec=/opt/android-studio/bin/studio.sh
Icon=/opt/android-studio/bin/studio.png
Categories=Development;IDE;
Terminal=false

Save and exit. Then make it executable:

chmod +x /usr/share/applications/android-studio.desktop

Verify Installation

You can now open Android Studio from the Application menu or by running:

android-studio

2. Create and Run Dart Programs in Android Studio

Step 1: Install Dart Plugin in Android Studio

  1. Open Android Studio
  2. Go to Settings or Preferences:
    • Linux/Ubuntu: File → Settings
    • Or use shortcut: Ctrl + Alt + S
  3. Navigate to: Plugins
  4. Search for Dart in the Marketplace
  5. Click Install
  6. Also install Flutter plugin if you plan to use Flutter
  7. Restart Android Studio after installation

Step 2: Create a Dart Project

  1. On the Android Studio welcome screen, click: New Project → Dart
  2. Choose: Dart Console Application
  3. Specify the project location
  4. Set Dart SDK path (Android Studio may prompt to download if not installed)

Or manually install Dart using:

sudo apt update
sudo apt install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt update
sudo apt install dart

Step 3: Write and Run Dart Program

Example Dart Program:

void main() {
  print("Hello, Dart!");
}

To run:

  • Click the green play button in Android Studio, or
  • Open terminal and run: dart run

Step 4: Run Dart in Terminal

To check Dart installation:

dart --version

Run any .dart file directly:

dart hello.dart

3. Flutter Widgets: Text, Image, and Container

1. Text Widget

The Text widget is used to display text on the screen with customizable styling.

import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text("Text Widget Example")),
      body: Center(
        child: Text(
          "Hello, Flutter!",
          style: TextStyle(
            fontSize: 24,
            fontWeight: FontWeight.bold,
            color: Colors.blue,
          ),
        ),
      ),
    ),
  ));
}

2. Image Widget

The Image widget displays images from various sources including network, assets, and files.

import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text("Image Widget Example")),
      body: Center(
        child: Image.network(
          'https://picsum.photos/300/300',
          width: 300,
          height: 300,
          fit: BoxFit.cover,
          loadingBuilder: (context, child, loadingProgress) {
            if (loadingProgress == null) return child;
            return CircularProgressIndicator();
          },
          errorBuilder: (context, error, stackTrace) {
            return Text('Failed to load image');
          },
        ),
      ),
    ),
  ));
}

3. Container Widget

The Container widget is a versatile widget that can contain padding, margins, borders, background color, and other decorations.

import 'package:flutter/material.dart';

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text("Container Widget Example")),
      body: Center(
        child: Container(
          width: 200,
          height: 100,
          padding: EdgeInsets.all(16),
          margin: EdgeInsets.all(20),
          decoration: BoxDecoration(
            color: Colors.amber,
            borderRadius: BorderRadius.circular(15),
            boxShadow: [
              BoxShadow(
                color: Colors.grey,
                blurRadius: 5,
                offset: Offset(3, 3),
              ),
            ],
          ),
          child: Center(
            child: Text(
              "This is a Container",
              style: TextStyle(fontSize: 18, color: Colors.black),
            ),
          ),
        ),
      ),
    ),
  ));
}

4. Display Text and Image using Container

Create Flutter Project

Step 1: Open Android Studio

/opt/android-studio/bin/studio.sh

Step 2: Create New Flutter Project

  • ALT + F → New → New Flutter Project
  • Click Next, select Flutter from generators (left side)
  • Click Next → Enter project name
  • Click Create

Step 3: Open main.dart

Select lib/main.dart from your project (left side panel)

Complete Code Example

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text("Text & Image in Container")),
        body: Center(
          child: Container(
            padding: EdgeInsets.all(16),
            decoration: BoxDecoration(
              color: Colors.blue[50],
              borderRadius: BorderRadius.circular(12),
              border: Border.all(color: Colors.blue, width: 2),
            ),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                Image.network(
                  'https://upload.wikimedia.org/wikipedia/commons/thumb/4/41/Flag_of_India.svg/320px-Flag_of_India.svg.png',
                  width: 150,
                  height: 150,
                  fit: BoxFit.cover,
                ),
                SizedBox(height: 10),
                Text(
                  'Hello Flutter!',
                  style: TextStyle(
                    fontSize: 20,
                    fontWeight: FontWeight.bold,
                  ),
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

Replace the above code in main.dart file.

Troubleshooting: If You Encounter Errors

Error 1: Missing build dependencies

sudo apt update
sudo apt install cmake ninja-build pkg-config libgtk-3-dev libblkid-dev liblzma-dev

Error 2: Missing build tools

sudo apt update
sudo apt install build-essential clang

After installing the required dependencies, run the program again.

Key Takeaways

  • Android Studio can be installed via Snap for quick setup or manually for more control
  • Dart and Flutter plugins are essential for Flutter development in Android Studio
  • Text widget is used for displaying styled text in Flutter applications
  • Image widget supports multiple sources: network, assets, files, and memory
  • Container widget provides layout, decoration, and positioning capabilities
  • Column widget arranges children vertically, useful for stacking elements
  • BoxDecoration allows styling containers with colors, borders, and shadows
  • Always install required Linux dependencies before running Flutter applications
