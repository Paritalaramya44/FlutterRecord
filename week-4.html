Week 4 — Flutter Layouts: Row, Column & Stack

Date: December 14, 2025

Learning Objectives

  1. Understand and implement Row layout for horizontal arrangement
  2. Understand and implement Column layout for vertical arrangement
  3. Understand and implement Stack layout for overlapping widgets
  4. Combine Row, Column, and Stack layouts in a single application
  5. Navigate between different layout examples using Flutter navigation

1. Row Layout Example

The Row widget arranges its children horizontally in a single line. It's useful for creating horizontal layouts like navigation bars, action buttons, or icon rows.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Row Layout Example',
      theme: ThemeData(primarySwatch: Colors.blue),
      home: RowExample(), // Home screen is RowExample
    );
  }
}

class RowExample extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text("Row Layout Example"),
        centerTitle: true,
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Row(
              mainAxisAlignment: MainAxisAlignment.spaceAround, 
              children: [
                Icon(Icons.home, size: 60, color: Colors.blue),
                Icon(Icons.star, size: 60, color: Colors.orange),
                Icon(Icons.settings, size: 60, color: Colors.green),
              ],
            ),
            SizedBox(height: 30),
            Text(
              'Name: Prashanth kumar',
              style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
            ),
            SizedBox(height: 10),
            Text(
              'Roll No: 238W1A12D4',
              style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
            ),
          ],
        ),
      ),
    );
  }
}

Key Concepts:

  • Row Widget: Arranges children horizontally
  • mainAxisAlignment: Controls spacing along the horizontal axis (spaceAround, spaceEvenly, center, etc.)
  • children: List of widgets to display in the row

2. Column Layout Example

The Column widget arranges its children vertically in a single column. It's ideal for stacking elements like forms, lists, or content sections.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Column Layout Example',
      theme: ThemeData(primarySwatch: Colors.blue),
      home: ColumnExample(), // Home screen is ColumnExample
    );
  }
}

class ColumnExample extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text("Column Layout Example"),
        centerTitle: true,
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.spaceEvenly, // Distribute evenly
          children: [
            Text(
              "Flutter",
              style: TextStyle(fontSize: 30, fontWeight: FontWeight.bold),
            ),
            ElevatedButton(
              onPressed: () {},
              child: Text("Click Me"),
            ),
            Icon(Icons.thumb_up, size: 60, color: Colors.purple),
            SizedBox(height: 20),
            Text(
              "Name: Prashanth kumar",
              style: TextStyle(fontSize: 20, fontWeight: FontWeight.w600),
            ),
            Text(
              "Roll No: 238W1A12D4",
              style: TextStyle(fontSize: 20, fontWeight: FontWeight.w600),
            ),
          ],
        ),
      ),
    );
  }
}

Key Concepts:

  • Column Widget: Arranges children vertically
  • mainAxisAlignment: Controls spacing along the vertical axis
  • spaceEvenly: Distributes space evenly between and around children

3. Stack Layout Example

The Stack widget allows you to overlay widgets on top of each other. It's perfect for creating layered UIs, badges, or complex designs where elements need to overlap.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Stack Layout Example',
      theme: ThemeData(primarySwatch: Colors.blue),
      home: StackExample(), // Home screen is StackExample
    );
  }
}

class StackExample extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text("Stack Layout Example"),
        centerTitle: true,
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Stack(
              alignment: Alignment.center, // Center all children
              children: [
                Container(width: 200, height: 200, color: Colors.blue),
                Container(width: 150, height: 150, color: Colors.red),
                Container(width: 100, height: 100, color: Colors.green),
                Text(
                  "Stacked",
                  style: TextStyle(color: Colors.white, fontSize: 20, fontWeight: FontWeight.bold),
                ),
              ],
            ),
            SizedBox(height: 30),
            Text(
              "Name: Prashanth kumar",
              style: TextStyle(fontSize: 20, fontWeight: FontWeight.w600),
            ),
            SizedBox(height: 10),
            Text(
              "Roll No: 238W1A12D4",
              style: TextStyle(fontSize: 20, fontWeight: FontWeight.w600),
            ),
          ],
        ),
      ),
    );
  }
}

Key Concepts:

  • Stack Widget: Overlays children on top of each other
  • alignment: Controls how children are positioned within the stack
  • Z-index: Children are stacked in the order they appear in the list (first child at bottom)

4. Row, Column & Stack in One App

This comprehensive example demonstrates navigation between different layout types using Flutter's Navigator. The app includes a home screen with buttons to navigate to each layout example.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Row, Column & Stack Demo',
      theme: ThemeData(primarySwatch: Colors.blue),
      home: HomeScreen(), // Start from menu
    );
  }
}

class HomeScreen extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Flutter Layouts")),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text(
              "Name: Prashanth kumar",
              style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
            ),
            SizedBox(height: 10),
            Text(
              "Roll No: 238W1A12D4",
              style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
            ),
            SizedBox(height: 40),
            ElevatedButton(
              child: Text("Row Layout"),
              onPressed: () {
                Navigator.push(
                  context,
                  MaterialPageRoute(builder: (context) => RowExample()),
                );
              },
            ),
            SizedBox(height: 20),
            ElevatedButton(
              child: Text("Column Layout"),
              onPressed: () {
                Navigator.push(
                  context,
                  MaterialPageRoute(builder: (context) => ColumnExample()),
                );
              },
            ),
            SizedBox(height: 20),
            ElevatedButton(
              child: Text("Stack Layout"),
              onPressed: () {
                Navigator.push(
                  context,
                  MaterialPageRoute(builder: (context) => StackExample()),
                );
              },
            ),
          ],
        ),
      ),
    );
  }
}

Navigation Concepts:

  • Navigator.push: Navigate to a new screen
  • MaterialPageRoute: Creates a route with Material Design transitions
  • context: BuildContext required for navigation
  • SizedBox: Creates spacing between buttons

Key Takeaways

  • Row widget arranges children horizontally, perfect for navigation bars and icon rows
  • Column widget arranges children vertically, ideal for forms and content stacking
  • Stack widget overlays children, useful for badges, backgrounds, and layered designs
  • mainAxisAlignment controls spacing and alignment along the primary axis
  • crossAxisAlignment controls alignment along the secondary axis
  • Navigator.push enables navigation between screens in Flutter applications
  • MaterialPageRoute provides Material Design page transitions
  • Combining layouts creates complex and responsive user interfaces
